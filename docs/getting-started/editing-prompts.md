# How to Edit Your Prompts

## Overview

Prompt Manage makes it easy to edit and refine your prompts after you've created them. Whether you need to fix a typo, update the content, or change settings, editing is quick and straightforward.

## Quick Start

1. Go to your [Dashboard](https://promptmanage.com/dashboard)
2. Find the prompt you want to edit
3. Click the three-dot menu (⋮) on the prompt card
4. Select "Edit"
5. Make your changes
6. Click "Update Prompt"

## Step-by-Step Guide

### 1. Access Your Dashboard

Navigate to your dashboard where all your prompts are displayed. You can access it by:

- Clicking "Dashboard" in the top navigation
- Going directly to `/dashboard`

### 2. Locate the Prompt

Find the prompt you want to edit. You can:

- **Scroll** through your prompts
- **Search** using the search bar
- **Filter** by tags, model, or visibility
- **Sort** by date, name, or other criteria

### 3. Open the Edit Menu

On each prompt card, you'll see a three-dot menu (⋮) in the top-right corner. Click it to reveal options including:

- View
- Edit
- Duplicate
- Delete
- Share

### 4. Click "Edit"

Select the "Edit" option from the menu. This will open the edit form with your prompt's current information pre-filled.

### 5. Make Your Changes

The edit form includes all the fields you can modify:

#### Basic Information

- **Name**: The title of your prompt
- **Prompt Text**: The actual prompt content
- **Model**: The AI model this prompt is designed for (GPT-4, Claude, Gemini, etc.)

#### Organization

- **Tags**: Keywords to help you find and organize prompts
- **Collection**: Assign to a collection (optional)
- **Category**: Categorize your prompt

#### Settings

- **Visibility**:
  - Private: Only you can see it
  - Team: Your team members can see it
  - Public: Anyone can see it in the directory
- **Description**: Additional context about the prompt
- **Variables**: Define placeholders in your prompt

### 6. Save Your Changes

Once you've made your edits:

1. Review your changes
2. Click the "Update Prompt" button
3. Wait for the success message

You'll see a confirmation toast notification: "Prompt Updated - [Your prompt name] has been successfully updated."

## What You Can Edit

### ✅ Editable Fields

- **Prompt Name**: Change the title anytime
- **Prompt Text**: Modify the actual prompt content
- **Model Selection**: Switch to a different AI model
- **Tags**: Add, remove, or change tags
- **Visibility**: Change who can see your prompt
- **Description**: Update the description
- **Collection**: Move to a different collection
- **Variables**: Add or modify prompt variables

### ❌ Non-Editable Fields

- **Created Date**: This is set when you first create the prompt
- **Prompt ID**: The unique identifier cannot be changed
- **Creator**: The original author cannot be changed

## Tips for Editing Prompts

### Best Practices

**1. Test After Editing**

- Always test your prompt after making changes
- Verify it still produces the desired output
- Check that variables still work correctly

**2. Use Version Control**

- Consider duplicating before major edits
- Keep a backup of working prompts
- Use descriptive names for different versions

**3. Update Tags and Metadata**

- Keep tags current and relevant
- Update descriptions when you change functionality
- Adjust the model if you optimize for a different AI

**4. Maintain Consistency**

- Use consistent formatting
- Follow your naming conventions
- Keep related prompts organized

### Common Editing Scenarios

**Fixing Typos**

1. Open the edit form
2. Correct the typo in the prompt text
3. Save immediately

**Optimizing for a Different Model**

1. Duplicate the original prompt (for backup)
2. Edit the copy
3. Change the model selection
4. Adjust the prompt text for the new model
5. Test thoroughly before saving

**Adding Variables**

1. Edit the prompt
2. Add variable placeholders (e.g., `{{variable_name}}`)
3. Define the variables in the Variables section
4. Save and test

**Changing Visibility**

1. Edit the prompt
2. Select new visibility setting
3. Save changes
4. Prompt immediately reflects new visibility

## Keyboard Shortcuts

While editing, you can use these shortcuts:

- `Cmd/Ctrl + S`: Save changes (in most browsers)
- `Esc`: Close the edit dialog
- `Tab`: Navigate between fields

## Troubleshooting

### Changes Not Saving

**Problem**: You click "Update Prompt" but changes don't save.

**Solutions**:

- Check your internet connection
- Ensure you're still logged in
- Try refreshing the page and editing again
- Clear your browser cache

### Edit Button Not Appearing

**Problem**: You can't find the edit option.

**Solutions**:

- Make sure you're the owner of the prompt
- Check that you're logged in
- Verify you have edit permissions (for team prompts)
- Try refreshing the page

### Form Not Pre-filling

**Problem**: The edit form opens but fields are empty.

**Solutions**:

- Refresh the page and try again
- Check your browser console for errors
- Try a different browser
- Contact support if the issue persists

### Permission Errors

**Problem**: You get a "Permission Denied" error.

**Solutions**:

- Verify you own the prompt or have team access
- Check that your account is active
- Ensure you're logged in with the correct account
- Contact your team admin for team prompts

## Frequently Asked Questions

### Can I undo changes after saving?

Currently, there's no built-in undo feature. We recommend duplicating important prompts before making major changes. Version history is coming soon.

### Can other people edit my prompts?

- **Private prompts**: Only you can edit
- **Team prompts**: Team members with edit permissions can edit
- **Public prompts**: Only you can edit (others can duplicate)

### What happens to shared links when I edit?

Shared links continue to work and will show the updated version of your prompt.

### Can I edit prompts on mobile?

Yes, the edit interface is fully responsive and works on mobile devices.

### Is there a limit to how many times I can edit?

No, you can edit your prompts as many times as you need.

### Can I edit prompts I've duplicated from others?

Yes, once you duplicate a prompt, you own the copy and can edit it freely.
